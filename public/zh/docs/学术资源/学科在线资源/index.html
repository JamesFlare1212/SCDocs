    加入我们 #

    你有两种贡献的方式，修正错误，以及完善内容。

    如果你希望提交改动，请以Pull Request的形式提出分支合并请求，而不是试图直接在主分支提交Commit。

    修正错误 #

    如需修正错误，你需要在本项目的仓库提出Issue。

    在提交 PR（Pull Request）之前，我希望你充分在Issue中讨论。

    完善内容 #

    如果你想提出新的想法，我推荐你在Discussions提出，然后进行讨论，而不是提出Issue，因为这不是错误，而是改进。

    如果你有不错的内容，或者知道优质的创作者，欢迎投稿。
